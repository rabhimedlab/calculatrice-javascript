# Calculatrice JavaScript

Une calculatrice simple développée en HTML, CSS et JavaScript.

## Fonctionnalités

- Addition
- Soustraction
- Multiplication
- Division
- Validation des champs
- Gestion de la division par zéro
- Historique des calculs
- Bouton Effacer
- Mode sombre
- Raccourcis clavier

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
